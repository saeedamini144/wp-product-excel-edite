== Description ==

WP Product Excel Edite یک افزونه حرفه‌ای برای مدیریت سریع و دقیق محصولات ووکامرس از طریق فایل‌های Excel (XLSX) است.  
این افزونه امکان **خروجی گرفتن از محصولات** و **ایمپورت دوباره همان فایل پس از ویرایش** را فراهم می‌کند و به‌صورت خودکار قیمت‌ها، موجودی، وضعیت موجودی و تخفیف‌ها را بروزرسانی می‌کند.

این افزونه مخصوص فروشگاه‌هایی طراحی شده که تعداد محصولات زیاد دارند و نیاز دارند قیمت‌ها و موجودی‌ها را به‌صورت گروهی و سریع بروزرسانی کنند.

---

== Features ==

- خروجی گرفتن از محصولات ووکامرس در قالب فایل XLSX
- ایمپورت فایل XLSX و بروزرسانی خودکار محصولات
- بروزرسانی قیمت عادی (Regular Price)
- بروزرسانی قیمت فروش ویژه (Sale Price)
- تنظیم تاریخ شروع و پایان تخفیف
- بروزرسانی موجودی انبار (Stock Quantity)
- تنظیم خودکار وضعیت موجودی (instock / outofstock)
- فعال/غیرفعال کردن مدیریت موجودی بر اساس مقدار موجودی
- ثبت محصولات پیدا نشده در فایل اکسل
- نمایش لیست محصولات پیدا نشده در صفحه تنظیمات افزونه
- صفحه‌بندی لیست محصولات پیدا نشده
- سازگار با PHP 7.4 تا 8.1
- استفاده از PhpSpreadsheet نسخه سازگار (1.25)

== How It Works ==

### 1. Export Products
در صفحه تنظیمات افزونه، بخش "خروجی Xlsx محصولات" وجود دارد.  
می‌توانید دسته‌بندی موردنظر را انتخاب کرده و فایل Excel محصولات را دانلود کنید.

### 2. Edit Excel File
فایل خروجی را در Excel باز کنید و موارد زیر را ویرایش کنید:

- قیمت عادی
- قیمت فروش ویژه
- تاریخ پایان تخفیف
- تعداد موجودی
- وضعیت موجودی (instock / outofstock)

### 3. Import Edited File
فایل ویرایش‌شده را در بخش "ایمپورت Xlsx" آپلود کنید.  
افزونه به‌صورت خودکار:

- محصول را فقط بر اساس **SKU** پیدا می‌کند
- قیمت‌ها را بروزرسانی می‌کند
- تخفیف‌ها را تنظیم می‌کند
- موجودی و وضعیت موجودی را اصلاح می‌کند
- مدیریت موجودی را بر اساس مقدار موجودی فعال/غیرفعال می‌کند

### 4. Not Found Products
اگر محصولی با SKU واردشده پیدا نشود:

- افزونه آن را در لیست "محصولات یافت نشد" ذخیره می‌کند
- این لیست در صفحه تنظیمات افزونه نمایش داده می‌شود
- لیست دارای صفحه‌بندی است
- امکان پاک‌سازی لیست وجود دارد

== Stock Logic ==

### ✔ اگر موجودی > 0:
- manage_stock = true  
- stock_status = instock  

### ✔ اگر موجودی = 0:
- manage_stock = false  
- stock_status = outofstock  

### ✔ اگر در فایل اکسل مقدار stock_status نوشته شده باشد:
- instock → مدیریت موجودی فعال، موجودی حداقل 1  
- outofstock → مدیریت موجودی غیرفعال، موجودی 0

== Installation ==

1. افزونه را در مسیر `/wp-content/plugins/` آپلود کنید.
2. افزونه را فعال کنید.
3. در منوی مدیریت، وارد بخش **Product Excel Exporter** شوید.
4. از بخش خروجی، فایل محصولات را دانلود کنید.
5. فایل را ویرایش کرده و دوباره ایمپورت کنید.
